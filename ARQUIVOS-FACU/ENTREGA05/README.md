# 💻 Sistema Inteligente de Estudos com IA


## 📝 Descrição do Projeto


Este projeto consiste em uma plataforma inteligente de assistência acadêmica desenvolvida
com foco em produtividade, aprendizado personalizado e integração com Inteligência Artificia
l. O sistema foi inspirado em ferramentas modernas de apoio educacional e evoluído com rec
ursos próprios, utilizando IA generativa e serviços do Firebase para oferecer uma experiênci
a completa, segura e escalável.
Desenvolvido como parte da disciplina de **Inteligência Artificial (2024.1)**, o sistema permit
e que usuários realizem perguntas utilizando texto, voz ou imagens, recebendo respostas ins
tantâneas e contextualizadas. A aplicação utiliza modelos de IA avançados para resolver que
stões, gerar conteúdos, sugerir ideias e auxiliar em diversas áreas do conhecimento.
O projeto foi modernizado com identidade visual própria, utilizando conceitos de **Dark Mode
Minimalista** e componentes inspirados em Glassmorfismo, proporcionando uma interface
moderna e intuitiva.
*Figura 1: Dashboard principal do sistema exibindo funcionalidades inteligentes e área de int
eração com IA.*

## 🚀 Tecnologias Utilizadas

* **Linguagem:** Python 3.10 e JavaScript
* **Bibliotecas:** Firebase SDK, Pandas, Scikit-learn, TensorFlow
* **Ferramentas:** Google AI Studio, Firebase, Google Colab, Jupyter Notebook
* **Banco de Dados:** Cloud Firestore
* **Hospedagem:** Firebase Hosting
* **Autenticação:** Firebase Authentication

## 📊 Resultados e Aprendizados

O projeto alcançou resultados sólidos em ambiente de testes, demonstrando eficiência na int
egração entre IA e aplicações web modernas.
* **Integração com IA Generativa:** Implementação de respostas inteligentes com suporte a t
exto, voz e imagens.
* **Sistema de Login Inteligente:** Utilização do Firebase Authentication para gerenciamento
seguro de usuários.
* **Armazenamento em Nuvem:** Histórico de consultas salvo no Cloud Firestore.
* **Analytics em Tempo Real:** Monitoramento das funcionalidades mais utilizadas pelos usu
ários.
* **Diferenciais Competitivos Implementados:**
* Exportação de relatórios personalizados.
* Sistema de filtros avançados para organização das consultas.
* Proteção de privacidade e armazenamento seguro.
* Configuração remota de interface utilizando Firebase Remote Config.
Além da implementação técnica, o projeto proporcionou aprendizado aprofundado em:
* Engenharia reversa assistida por IA.
* Desenvolvimento de aplicações escaláveis utilizando Firebase.
* Integração entre modelos de IA e interfaces web modernas.
* Ética e originalidade no desenvolvimento de software com IA.
## 🔧 Como Executar

1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Configure as credenciais do Firebase.
4. Execute o comando: `python main.py`.
5. Acesse a aplicação utilizando o Firebase Hosting.
