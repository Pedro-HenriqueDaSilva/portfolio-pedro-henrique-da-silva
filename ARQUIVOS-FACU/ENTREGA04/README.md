#  💻 Gerador Dinâmico de QR Codes (Projeto Vibe Coding)

## 📝  Descrição do Projeto
Este projeto consiste num motor de geração de QR Codes desenvolvido de forma ágil
através do ambiente Google AI Studio (crie-qr-code). O objetivo principal é mitigar a
dificuldade técnica de criar ferramentas utilitárias personalizadas, oferecendo uma
interface web funcional onde os utilizadores podem converter links, textos e informações
de contacto em códigos digitalizáveis instantaneamente.
Desenvolvido utilizando o novo paradigma de Vibe Coding (2024.1), o sistema utiliza a
inteligência artificial do Gemini para orquestrar o código-fonte (HTML, CSS e JavaScript),
permitindo que a lógica de geração de QR Codes seja integrada de forma fluida sem a
necessidade de configurações complexas de servidor.
*Figura 1: Interface do App 'crie-qr-code' no Google AI Studio, exibindo o campo de entrada e o
QR Code gerado em tempo real.*
## ⌨️ Tecnologias Utilizadas
* **Linguagem:** JavaScript (ES6+), HTML5, CSS3
* **Bibliotecas:** QRCode.js ou APIs de geração de imagem (Google Charts API)
* **Ferramentas:** Google AI Studio (Vibe Code), Gemini API, Vercel/GitHub (para deploy
opcional)
## 📚 Resultados e Aprendizados
O projeto demonstrou a eficiência da criação de apps "no-code/low-code" assistida por IA:
* **O modelo atingiu 100% de funcionalidade** na conversão de URLs para imagens
digitalizáveis em tempo real.
* **Redução de Ruído:** Aprendi a utilizar prompts no AI Studio para limpar o design da
interface, removendo elementos desnecessários e focando na experiência do utilizador
(UX).
* **Visualização de Fluxo:** Implementei uma lógica de feedback visual que indica ao
utilizador quando o QR Code está pronto para download ou partilha.
