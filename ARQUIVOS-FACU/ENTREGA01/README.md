
# 🎬 Sistema de Geração Criativa com Inteligência Artificial

## 📝 Descrição do Projeto
Este projeto apresenta um sistema experimental de geração de conteúdo utilizando Inteligência
Artificial, com foco em criação de textos e imagens a partir de prompts personalizados. O objetivo
principal foi explorar a capacidade de modelos generativos em interpretar instruções criativas e
produzir resultados coerentes, detalhados e visualmente impactantes.
Durante os testes realizados, foram utilizados prompts como:
* "Um astronauta estilo barroco tocando violoncelo em Marte."
* "Um e-mail formal de desculpas de um pirata para um rei."
O sistema foi capaz de gerar imagens artísticas em diferentes estilos visuais e também produzir textos
contextualizados, demonstrando criatividade, adaptação de linguagem e interpretação semântica.

##  🚀 Tecnologias Utilizadas
* **Inteligência Artificial:** Modelos generativos de texto e imagem
* **Linguagem:** Python 3.10
* **Bibliotecas:** OpenAI API, Pillow, Matplotlib
* **Ferramentas:** Jupyter Notebook, Google Colab
## 📊 Resultados e Aprendizados

O projeto apresentou resultados satisfatórios em ambiente de testes, destacando a eficiência dos
modelos generativos em diferentes cenários criativos.

* **Geração de Conteúdo:** Produção automática de textos contextualizados e imagens artísticas.*
  
* **Interpretação de Prompts:** Aprendizado sobre construção de prompts mais eficientes e
detalhados.*
**Criatividade Computacional:** Exploração de técnicas de IA para geração visual e textual.*
 
* **Qualidade Visual:** Desenvolvimento de imagens com estilos variados e boa composição artística.*
